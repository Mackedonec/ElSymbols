Symbol Name Explanation
Switch (closed) Замкнений перемикач. Контакти з’єднані, тому струм може проходити через коло.
Switch (open) Розімкнений перемикач. Контакти розділені, тому струм не проходить.
Cell Окремий гальванічний елемент, який створює електричну напругу.
Resistor Резистор обмежує силу струму в електричному колі.
Capacitor Конденсатор накопичує електричний заряд і тимчасово зберігає енергію.
Inductor Котушка індуктивності накопичує енергію у магнітному полі.
Wire Провідник, який з’єднує елементи електричної схеми.
Resistor (rectangle) Європейський варіант позначення резистора.
Battery Батарея складається з кількох елементів (cell) і забезпечує більшу напругу.
Fuse Запобіжник розриває коло при надмірному струмі для захисту пристроїв.
Variable Resistor Змінний резистор дозволяє регулювати опір у схемі.
Thermistor Резистор, опір якого змінюється залежно від температури.
Ammeter Прилад для вимірювання сили струму.
Voltmeter Прилад для вимірювання електричної напруги.
Motor Електродвигун, що перетворює електричну енергію на механічний рух.
Diode Діод пропускає струм лише в одному напрямку.
LDR Фоторезистор, опір якого змінюється залежно від освітлення.
Ground Спільна точка нульового потенціалу у схемі.
Buzzer Електронний звуковий сигналізатор.
Lamp Лампа, яка світиться при проходженні струму.
Transistor Напівпровідниковий компонент для підсилення сигналів або перемикання.
LED Світлодіод (Light Emitting Diode). Випромінює світло, коли через нього проходить електричний струм.
Push Button (NO) Кнопка з нормально розімкнутими контактами (Normally Open). Коло замикається лише під час натискання.
Potentiometer Потенціометр — змінний резистор з трьома виводами, який використовується для регулювання напруги або сигналу.
Junction Точка з'єднання проводів. Показує, що провідники електрично з'єднані між собою.
Crossing Wires Перехрещення проводів без електричного з'єднання.
NPN Transistor NPN транзистор — використовується для підсилення сигналів або як електронний перемикач.
PNP Transistor PNP транзистор — працює подібно до NPN, але струм керування має протилежну полярність.
Operational Amplifier Операційний підсилювач (Op-Amp). Використовується для підсилення сигналів та обробки аналогових даних.
Crystal Oscillator Кварцовий резонатор. Використовується для створення стабільної частоти в електронних пристроях.
Relay Реле — електромеханічний перемикач, який керується електричним сигналом.
Integrated Circuit (IC) Інтегральна схема — електронний компонент, що містить багато транзисторів та інших елементів на одному кристалі.
Microcontroller Мікроконтролер — програмований чип, що містить процесор, пам’ять та периферію.
AND Gate Логічний елемент AND. Вихід активний лише тоді, коли всі входи активні.
OR Gate Логічний елемент OR. Вихід активний, якщо активний хоча б один вхід.
NOT Gate Логічний інвертор. Вихід має протилежне значення до входу.
NAND Gate Логічний елемент NAND — інверсія AND. Вихід вимкнений лише якщо всі входи активні.
NOR Gate Логічний елемент NOR — інверсія OR.
Connector / Header Роз’єм або контактна колодка для підключення проводів або інших плат.
Transformer Трансформатор — пристрій для передачі електричної енергії між колами через магнітне поле.
Antenna Антена — використовується для передачі або прийому радіосигналів.
